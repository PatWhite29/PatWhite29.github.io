Preguntas HTML:

1. What is HTML?

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de internet. Es utilizado para estructurar y presentar contenido en la web. HTML define la estructura de una página mediante etiquetas.

2. Explain the head and the body tags.

El head es una sección de la página que no se muestra en el navegador, pero que contiene información sobre el documento y enlaces a otros archivos. El body es la sección de la página que se muestra en el navegador y que contiene el contenido de la página.

3. Explain the section, article, aside, header, footer tag?

La etiqueta section define una sección en un documento HTML. La etiqueta article define un artículo en un documento HTML. La etiqueta aside define contenido que está relacionado con el contenido que lo rodea, pero que no es parte de él. La etiqueta header define una cabecera para el documento o una sección. La etiqueta footer define un pie de página para el documento o una sección.

Preguntas CSS:

1. What is CSS?

CSS is a language of design for webpages to add style, add fonts, colors, margins and visual elements. to an HTML Document.

2.What is the purpose of CSS?
